# Superstore Sales Dashboard using Power BI

## Overview

This project focuses on the development of an interactive sales dashboard using Microsoft Power BI. The dashboard was created using the Superstore dataset to analyze and visualize important business insights such as sales performance, profit analysis, regional sales distribution, customer contribution, and shipping trends. The purpose of this dashboard is to transform raw sales data into meaningful visual information that helps in business decision-making.

## Tools and Technologies Used

* Microsoft Power BI
* Superstore CSV Dataset
* Data Visualization Techniques
* Interactive Dashboard Design

## Dashboard Features

The dashboard contains multiple interactive visualizations and KPI cards that provide a complete overview of the sales data. The top section of the dashboard includes key performance indicators such as:

* Total Sales
* Total Profit
* Total Orders
* Average Discount

These KPI cards help users quickly understand the overall business performance.

The dashboard also includes several charts and graphs:

### 1. Sales by Category

A bar chart is used to compare sales among different product categories such as Technology, Furniture, and Office Supplies. This helps identify the category generating the highest revenue.

### 2. Profit by Sub-Category

This chart visualizes profit earned from different sub-categories of products. It helps in understanding which products are profitable and which products generate losses.

### 3. Sales by Region

A pie chart is used to show the percentage contribution of different regions such as West, East, Central, and South. This provides insights into regional business performance.

### 4. Sales by Order Date

A line chart is used to display sales trends over time. This helps in identifying periods of high and low sales activity.

### 5. Sales by Customer Name

A bar chart highlights the top customers contributing the highest sales. This helps businesses identify valuable customers.

### 6. Sales by Ship Mode

A donut chart represents sales distribution based on shipping modes such as Standard Class, Second Class, First Class, and Same Day delivery.

## Interactive Features

The dashboard includes slicers for filtering data by:

* Category
* Region

These filters make the dashboard interactive and allow users to analyze specific sections of data dynamically.

## Insights Gained

From the dashboard analysis, several useful insights were identified:

* Technology category generated the highest sales.
* The West region contributed the highest percentage of sales.
* Some product sub-categories produced low or negative profits.
* Standard Class was the most commonly used shipping mode.
* Certain customers contributed significantly more sales compared to others.
